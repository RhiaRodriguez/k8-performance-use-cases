1. Download Jmeter first https://jmeter.apache.org/download_jmeter.cgi 
2. Download Java SDK 8 up https://www.oracle.com/ph/java/technologies/javase/javase-jdk8-downloads.html
3. Download Jmeter jar file (upload here)
4. Transfer the JMX file and JAR file in the "BIN" folder
5. Open the Jmeter meter using the JAR file in the BIN folder
6. Open the either GlasswallPOCJsonreport or GlasswallPOCwithloadpattern JMX file
7. Click "Run" button
